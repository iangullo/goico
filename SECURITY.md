# Security Policy

## Supported Versions

Since Goico is - for the moment - a personal hobby project I maintain in my spare time, only the last published version is (more or less) supported.
| Version | Supported          |
| ------- | ------------------ |
| 0.0.1   | :white_check_mark: |
| < 0.0.1 | :x:                |

## Reporting a Vulnerability

Post a comment in the release notice or open an issue stating version and providing as much detail as you can on the nature of the vulnerability..
